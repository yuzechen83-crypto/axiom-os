# axiom-core-proprietary

Axiom-OS 核心算法私有包。需授权获取。

## 安装

```bash
pip install -e .
```

或从项目根目录：

```bash
pip install -e ./axiom_core_proprietary
```

## 包含模块

- RCLN (Residual Coupler Linking Neuron)
- Discovery Engine
- Hippocampus
- Wind Hard Core / Adaptive Hard Core
- SPNN-Evo Coach
- Bundle Field (元轴丛场)
- Partition (智能分区)

## 私有仓库

本包应推送到私有 Git 仓库，仅供授权用户使用。

```bash
cd axiom_core_proprietary
git init
git add .
git commit -m "Proprietary core v1.0"
git remote add origin <your-private-repo-url>
git push -u origin main
```
